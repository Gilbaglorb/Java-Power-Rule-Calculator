Concept:
This is a power rule calculator for single terms. In calculus, the first method of taking derivatives you learn is the power rule (excluding the limit definition of the derivative).

Functionality:
The user is given a welcome message and prompted to enter a single term in terms of x.
The user inputs the term.
The program does the power rule on this term.
The program outputs the updated term.

Valid inputs: 
  -Terms containing "x"
    -"x"
    -"ax"
    -"ax^n"
    -"x^n"
  -Any constant

Let me know what needs to be fixed!

Note: values are int, not long, there is a limit to the numbers you can use.
